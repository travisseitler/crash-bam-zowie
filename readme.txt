Easily manage and publish your webcomics with WordPress.

== Description ==

This plugin makes it easy to manage one or more webcomics series on your WordPress site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/crash-bam-zowie` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. The Webcomics->Webcomic Settings screen serves as a summary of all series (or issues/chapters, if you only run a single series)
1. Use the Webcomics->Series & Issues screen to add series and their issues or chapters (if any)
1. You can add a new page at Webcomics->Add Comic Page
